# Gait Asymmetry in Parkinson's Disease

This repository contains the analysis pipeline used to compare
different asymmetry formulations applied to spatio-temporal
gait parameters in Parkinson’s disease.

The analysis includes:

• mathematical structure of asymmetry metrics
• discrimination between PD and healthy subjects
• association with Hoehn & Yahr progression
• comparison between asymmetry formulas and gait variables

The dataset is not publicly available due to privacy restrictions.
